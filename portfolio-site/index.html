AI Test Studio

AI Test Studio is a new unified testing UI driving a fundamental shift in software testing — from fragmented, skill‑dependent processes to an AI‑driven, autonomous testing fabric that unifies functional, performance, UX, accessibility, and API testing into a single workflow. It embodies the next evolution of Perforce's AI‑driven “shift‑left” testing vision.

Product case study

BlazeMeter Test Data Pro

BlazeMeter Test Data Pro is an AI‑powered tool that generates compliant synthetic test data on demand — through simple natural‑language prompts — instead of using risky production data. It scales from a few records to millions across functional, performance, and mobile testing, enabling earlier validation, broader coverage, and faster releases.

I was involved from the beginning of the design process for this feature, from ideation and research through to the first high‑fidelity mockups and user testing.

BlazeMeter Virtual Services

BlazeMeter Service Virtualization lets teams simulate unavailable or unstable systems — APIs, back‑end services, and third‑party dependencies — so testing never stalls waiting on missing components. Lightweight, on‑demand virtual services embed into CI/CD pipelines and replicate real‑world conditions such as slow responses and errors, enabling true shift‑left testing across web and mobile.

Blaze Bootstrap

When the Service Virtualization section of BlazeMeter — one of its main parts — was built from scratch, I created Blaze Bootstrap: a living CSS style guide that keeps the UI consistent and streamlines how components are developed and maintained.
